# iMove

将 USB 存储设备中的照片和视频传输到手机的 Android 应用。

## 功能

- 自动检测 USB 存储设备，支持热插拔
- 按日期分组浏览，支持筛选
- 单张或批量传输，支持队列管理
- 文件预览，支持缩放和滑动切换
- 支持中/英文，浅色/深色主题

## 环境要求

- Android 10+（API 29）
- 支持 OTG 的 Android 设备

## 技术栈

Kotlin · Jetpack Compose · Hilt · Room · DataStore · MVVM

## 构建

```bash
./gradlew assembleDebug
```
